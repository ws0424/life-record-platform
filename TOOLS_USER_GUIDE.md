# 生活小工具使用指南

## 🚀 快速开始

### 1. 访问工具首页
打开浏览器访问: http://localhost:3000/tools

你会看到 12 个工具卡片，目前已完成的工具有：
- ✅ 倒计时
- ✅ 待办清单
- ✅ 记账本

### 2. 使用倒计时

**创建倒计时**
1. 点击"倒计时"卡片或访问 http://localhost:3000/tools/countdown
2. 点击"创建倒计时"按钮
3. 填写信息：
   - 标题：例如"生日"、"考试"
   - 描述：可选
   - 目标日期：选择日期
   - 类型：选择类型（生日、纪念日、考试等）
   - 颜色：自定义颜色
4. 点击"确定"

**功能特点**
- 自动计算剩余天数
- 卡片式展示，一目了然
- 支持编辑和删除
- 按类型分类显示

### 3. 使用待办清单

**创建待办**
1. 点击"待办清单"卡片或访问 http://localhost:3000/tools/todo
2. 点击"创建待办"按钮
3. 填写信息：
   - 标题：要做的事情
   - 描述：详细说明（可选）
   - 优先级：低/中/高/紧急
   - 截止日期：可选
   - 分类：工作、学习、生活等
   - 标签：添加标签便于管理
4. 点击"确定"

**功能特点**
- 查看统计数据（总任务、待办、已完成、完成率）
- 一键完成/取消完成
- 按状态筛选（全部/待办/进行中/已完成）
- 优先级颜色标识
- 支持编辑和删除

**快捷操作**
- 点击复选框：快速标记完成/未完成
- 点击编辑按钮：修改待办信息
- 点击删除按钮：删除待办

### 4. 使用记账本

**记收入**
1. 点击"记账本"卡片或访问 http://localhost:3000/tools/expense
2. 点击"记收入"按钮（绿色）
3. 填写信息：
   - 标题：例如"工资"、"奖金"
   - 金额：输入金额
   - 日期：选择日期
   - 分类：工资/奖金/投资/礼物/其他
   - 备注：可选
   - 标签：可选
4. 点击"确定"

**记支出**
1. 点击"记支出"按钮（红色）
2. 填写信息：
   - 标题：例如"午餐"、"交通"
   - 金额：输入金额
   - 日期：选择日期
   - 分类：餐饮/交通/购物/娱乐/住房/医疗/教育/水电/其他
   - 备注：可选
   - 标签：可选
3. 点击"确定"

**功能特点**
- 查看统计数据（总收入、总支出、结余）
- 收入显示绿色，支出显示红色
- 按类型筛选（全部/收入/支出）
- 支持编辑和删除
- 金额自动格式化

## 💡 使用技巧

### 倒计时
- 使用不同颜色区分不同类型的倒计时
- 重要的日子可以选择醒目的颜色
- 定期检查即将到来的倒计时

### 待办清单
- 合理设置优先级，先做紧急重要的事
- 使用标签分类管理不同领域的任务
- 定期清理已完成的任务
- 利用完成率激励自己

### 记账本
- 每天记录当天的收支
- 使用标签标记特殊支出（如旅游、购物节）
- 定期查看统计数据，了解消费习惯
- 合理规划收支，保持正结余

## 🎨 界面说明

### 统一设计元素
- **渐变色标题**：每个工具都有独特的渐变色
- **卡片布局**：信息清晰，易于浏览
- **动画效果**：流畅的进入和悬停动画
- **响应式设计**：支持手机、平板、电脑

### 颜色含义
- **绿色**：成功、收入、完成
- **红色**：删除、支出、紧急
- **蓝色**：信息、中等优先级
- **橙色**：警告、高优先级
- **灰色**：次要信息、已取消

## 🔐 权限说明

所有工具都需要登录后才能使用：
- 未登录访问会自动跳转到登录页
- 每个用户只能看到和管理自己的数据
- 数据完全隔离，保护隐私
